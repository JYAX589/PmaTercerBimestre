Introducción a Git

¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite a los desarrolladores de software realizar un seguimiento de los cambios en el código fuente durante el desarrollo de software. Fue creado por Linus Torvalds en 2005.

Con Git, los desarrolladores pueden colaborar en un proyecto, ver el historial de cambios, revertir a versiones anteriores y mucho más. Git es esencial para el desarrollo de software moderno y es ampliamente utilizado en la industria.

Comandos de Git
1. git init

Inicializa un nuevo repositorio Git.

git init
2. git clone

Clona un repositorio existente.

git clone https://github.com/usuario/repo.git
3. git status

Muestra el estado de los cambios en el repositorio.

git status
4. git add

Añade archivos al área de preparación.

git add archivo.txt
5. git commit

Guarda los cambios en el repositorio.

git commit -m "Mensaje del commit"
6. git push

Envía los cambios al repositorio remoto.

git push origin main
7. git pull

Obtiene y fusiona cambios del repositorio remoto.

git pull origin main
8. git branch

Muestra las ramas existentes.

git branch
9. git checkout

Cambia a otra rama.

git checkout nombre_rama
10. git merge

Fusiona otra rama en la rama actual.

git merge nombre_rama
